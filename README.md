# Flexible-Link-end-mass
This is a computational Model focused on the gravity-induced free oscillations of a flexible link with a rigid payload
The MATLAB model is built entirely from first principles 
The model captures : 
1. Coupling of translational and rotational inertia at the payload
2. Boundary conditions including moment balance and slope continuity,
3. Natural frequencies and mode shapes through a characteristic determinant formulation.

Alongside , the same model has been simulated in Adams which uses Craig-Bampton Reduction method 
and a close agreement between the two solutions(Matlab and Adams) validate the physical fidelity of the variational formulation .
